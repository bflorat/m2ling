Link instance update

Administrators or actors with adequate authorizations can update a link instance.

If a link instance source is set to null.
Then the service should : FAIL with code NULL_ARGUMENT

If a link instance destination is set to null.
Then the service should : FAIL with code NULL_ARGUMENT

If a link instance source doesn't follow source component instance defined in its link,
Then the service should : FAIL with code LI_ILLEGAL_SOURCE_OR_DEST

If a link instance destination doesn't follow destination component instance defined in its link,
Then the service should : FAIL with code LI_ILLEGAL_SOURCE_OR_DEST
