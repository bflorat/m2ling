Link instance search

Administrators or actors with adequate authorizations can search for a link instance.

It is possible to retrieve all existing link instances :

Given several existing LIs
When we ask for LI listing
Then we get a number of LI >0.

It is possible to get a LI by id :

Given an existing link whose id=id_li_vp2_v2_l1_1
When we search for it by ID
Then we should be a link instance named link_vp2_v2_l1_1.
