Component search search

Administrators or actors with adequate authorizations can search a component.

It is possible to retrieve all existing components for a given view:

Given several existing components in id_view_vp_logical_Logical_BikesOnlineview
When we ask for components listing
Then we get a number of components >0.

It is possible to get components by non null id :

Given an existing component whose id=id_comp_view_Logical_BikesOnline_AdminGUI
When we search for it by ID
Then we should be a component named AdminGUI.
Same query with null id should FAIL with code NULL_ARGUMENT.

It is possible to retrieve all components and groups for a given CT:

Given some components of the same type and component groups with component type : id_ct_vp1_ct14
When we ask for components and groups
Then we get this output : [id=id_comp_vp1_test_comp_by_ct_1 name=comp_vp1_test_comp_by_ct_1 , id=id_comp_vp1_test_comp_by_ct_2 name=comp_vp1_test_comp_by_ct_2 , id=id_group_vp1_test_comp_by_ct_1 name=group_vp1_test_comp_by_ct_1 ].
