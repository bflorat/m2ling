View search

Administrators or actors with adequate authorizations can search a View.

It is possible to retrieve all existing Views :

Given several existing Views
When we ask for Views listing
Then we get a number of Views >0.

It is possible to get views by non null id :

Given an existing view whose id=id_vp1_v1
When we search for it by ID
Then we should be a View named vp1_v1.
Same query with null id should FAIL with code NULL_ARGUMENT.
