Rule update

Administrators or actors with adequate authorizations can update rules.

Every rule attribute, including its name can be changed except its id.

Given a rule whose id is id_rule1 and view point id id_vp1
When we update the view point with following attributes :
  • id=new rule name
  • id=new desc
  • id=new rationale
  • id=new exceptions
  • id=new comment
  • id=new tag1,new tag2
  • id=REFUSED
  • id=HIGH
Then we the viewpoint should now have the following characteristics : comment=new comment description=new desc drop= exceptions=new exceptions id=id_rule1 name=new rule name priority=HIGH rationale=new rationale status=REFUSED tags=new tag1,new tag2 vpID=id_vp1.
