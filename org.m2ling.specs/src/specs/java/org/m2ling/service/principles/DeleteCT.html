Component type deletion

CT20 : A CT can't be removed while at least a single component of this type exists in any view.

In this case, the service should : FAIL with code CANT_DELETE_CT_EXISTING_COMP
If the CT has no associated component of its type, the deletion service should : PASS

CT21 : A CT can't be removed while at least a single CT is bound to it.

In this case, the service should : FAIL with code CANT_DELETE_CT_EXISTING_BINDING
