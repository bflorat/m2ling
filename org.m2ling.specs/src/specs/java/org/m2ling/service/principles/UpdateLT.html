Link type update

Administrators or actors with adequate authorizations can update a LT (Link type).

LT30 : A LT source type can't be removed while at least a single link owns a source component of this CT in any view.

In this case id_vp1_link_type_1, the update service should : FAIL with code LT_EXISTING_LINK
If the LT has no associated link id_vp1_no_link of its type, the update service should : PASS

LT31 : A LT destination type can't be removed while at least a single link owns a source component of this CT in any view.

In this case id_vp1_link_type_1, the update service should : FAIL with code LT_EXISTING_LINK
If the LT has no associated link of its type, id_vp1_nolink the update service should : PASS

LT32 : A LT shouldn't wind up with zero source type after an update.

In this caseid_vp1_link_type_1, the update service should : FAIL with code LT_EXISTING_LINK

LT33 : A LT shouldn't wind up with zero destination type after an update.

In this case, id_vp1_link_type_1 the update service should : FAIL with code LT_EXISTING_LINK
