Rules creation

Administrators or actors with adequate authorizations can create a new rule against a given viewpoint.

Automated

Example

Given a view point named id_vp1, a new rule created with :

  • id=ruleid1
  • id=rule1
  • id=desc1
  • id=comment1
  • id=VALIDATED
  • id=HIGH
  • id=rationale1
  • id=exception1
  • id=tag1,tag2
  • id=1330347208000:SUGGESTED,1341347208345:VALIDATED
should have following characteristics : comment=comment1 description=desc1 exceptions=exception1 history={1330347208000=SUGGESTED, 1341347208345=VALIDATED} id=ruleid1 name=rule1 priority=HIGH rationale=rationale1 status=VALIDATED tags=tag1,tag2 vpID=id_vp1.
