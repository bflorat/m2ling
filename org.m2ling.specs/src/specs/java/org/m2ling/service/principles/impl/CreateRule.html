Rules creation

Administrators or actors with adequate authorizations can create a new rule against a given viewpoint.

Automated

Example

Given a view point named id_vp1, a new rule created with :

  • id=ruleid1
  • name=rule1
  • description=desc1
  • comment=comment1
  • status=VALIDATED
  • priority=HIGH
  • rationale=rationale1
  • exception=exception1
  • tags=tag1,tag2
should have following characteristics : comment=comment1 description=desc1 drop= exceptions=exception1 id=ruleid1 name=rule1 priority=HIGH rationale=rationale1 status=VALIDATED tags=tag1,tag2 vpID=id_vp1.
