Component type deletion

Administrators or actors with adequate authorizations can delete a CT (component type).

It is possible to retrieve all existing viewpoints :

Given several existing component type
When we ask for viewpoints listing
Then we get a number of ct >0.

It is possible to get viewpoints by id :

Given an existing CT whose id=id_comp_type_1
When we search for it by ID
Then we should be a viewpoint named comp_type_1.
