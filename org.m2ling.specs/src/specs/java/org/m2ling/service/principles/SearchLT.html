Link type search

Administrators or actors with adequate authorizations can search for a LT (Link type).

It is possible to retrieve all existing LTs :

Given several existing Link type
When we ask for lt listing
Then we get a number of lt >0.

It is possible to get LT by id :

Given an existing LT whose id=id_lt_vp1_link_type_1
When we search for it by ID
Then we should be a CT named lt_vp1_link_type_1.
