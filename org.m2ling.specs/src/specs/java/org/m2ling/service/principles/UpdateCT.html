Component type Component type update

Administrators or actors with adequate authorizations can Component type update a CT (component type).

Changes

Any change of the instantiation factor must be be compatible with the number of instances of component of this type and of component bound to the type.

Bound type following attributes can be changed : name, tags , comment, description

Bound type following attributes *can't* be changed : instantiation factor, reifiable flag

A reference can't be dropped while at least one component of the component type use it against another component.
