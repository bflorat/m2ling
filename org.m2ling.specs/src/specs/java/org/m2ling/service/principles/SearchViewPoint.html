Viewpoint search

Administrators or actors with adequate authorizations can search a viewpoint.

It is possible to retrieve all existing viewpoints :

Given five existing viewpoints vp1 and vp2
When we ask for viewpoints listing
Then we get 5 viewpoints.

It is possible to get viewpoints by id :

Given an existing viewpoint whose id=id_vp1
When we search for it by ID
Then we should be a viewpoint named vp1.

It is possible to get viewpoints by name :

Given an existing viewpoint whose name=vp1
When we search for it by name
Then we should be a viewpoint whose ID is id_vp1.
