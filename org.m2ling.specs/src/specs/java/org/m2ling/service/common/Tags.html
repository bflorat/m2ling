Tag principles

  • Most m2ling items have a tag and they can be set from most pages.
  • Search is case insensitive.
  • Search can mix tags like : foo,bar.
  • Most m2ling tools provide filters text fields used to filter by tags (even diagrams).

Apply tags against a taggable item

User can add through GUI one or more tags to taggable items. Tags are separated by comma. Spaces before or after the separator are ignored. a tag cannot contain the separator char.

Special cases

  • If tags are globally null, a FunctionalException is expected.
  • If a single tag contains a comma, a FunctionalException is expected.
  • If a single tag is void, a FunctionalException is expected.
  • If a single tag is null, a FunctionalException is expected.
