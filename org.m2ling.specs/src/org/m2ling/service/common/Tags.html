Tag principles

  • Most m2ling items have a tag
  • Search is case insensitive.
  • Search can mix tags like : foo,bar.
  • Most m2ling tools provide filters textfields used to filter by tags (even diagrams)

Apply tags against a taggeable item

User can add through GUI one or more tags to taggeable items. Tags are separated by coma. Spaces before or after the separator are ignored. a tag cannot contain the separator char.
