General GUI specifications

  • No auto-save (DRM, 21 feb).
  • Applications switching : top bar containing every available.
    • application for current user + "others" dropdown button.
  • A comments area in the bottom of the screen always visible.
  • a status line at the bottom of the screen.

GUI specifications syntax

We use this syntax for text-based GUI description in following specifications :
  • Tree item [right click menu] (left click effect)
  • [XXX] : textfield
  • + XXX : tree static item, * YYY : tree user-content item
  • @see YYY: same idea than YYY
  • #1 : form detailed in upcoming paragraph
  • <XXX> : label
  • [1] : comment described bellow
  • [[aa,bb]] : List
  • ?[] : combobox, ?[]+ : editable combobox
  • [] abc : chekbox
  • [[],[]] : table
